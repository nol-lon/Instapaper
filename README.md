# Obsidian Beige Theme

A warm, Instapaper-inspired theme for Obsidian that prioritizes readability and a calm writing experience.

## Features

✨ **Warm Beige Color Palette**
- Uniform beige background throughout the entire interface
- Subtle hover states with for visual feedback

📖 **Optimized for Reading**
- Times New Roman serif font for body text and headings
- 1.6 line-height for optimal readability
- Active line highlighting in editing mode

🎨 **Clean & Minimal Design**
- Floating/rounded tabs for a modern look
- Subtle borders and dividers
- Dark tooltips for better contrast


## Theme Modes

This theme includes two appearance modes:

- **Light Mode (Beige)** - The signature warm, Instapaper-inspired reading experience
- **Dark Mode (OLED)** - A standard dark theme for nighttime use


## Compatibility

- **Obsidian Version:** 0.16.0+
- **Tested Platforms:** macOS, Windows, Linux
- **Mobile Support:** Yes (may require minor adjustments)

### Development

To modify this theme:

1. Clone the repository
2. Edit `theme.css`
3. Reload Obsidian (Ctrl/Cmd + R) to see changes
4. Test in both light and dark modes
